This page provides a list of the FHIR artifacts defined as part of this implementation guide.

Structures: Resource Profiles

These define constraints on FHIR resources for systems conforming to this implementation guide.

Lab Order

A service request that initiates the need for the lab to collect the test sample
Presenting Symptoms

Existing symptoms which may help with diagnosis
Treatment dispensed or prescribed

To indicate whether Paxlovid has been issued to the individual
Specimen

The test sample that was collected for the initiated lab order
Recovered Or Symptoms Resolved

Has the individual fully recovered from symptoms associated with COVID19?
Lab Task

Assists with tracking the state of the lab order and its completion status
Practitioner

The healthcare professional who has been assigned to a given lab task
Vaccine Dose Ever Received

Has the individual received at least one shot of the vaccine at any point in time in the past?
Patient

The individual receiving COVID19 services
Immunization Tracking

Retrospectively capture COVID19 immunization data for vaccines administered at other organizations
Long Covid / Post-Covid

Does the indiviual continue to suffer from COVID19 health problems that are new, returning or ongoing conditons?
Diagnostic Report

A report as a result of the lab task being completed
Conditions or Comorbidity

What pre-existing conditions and/or comorbidities exist for the individual?
Encounter

A record of the individual’s visit at the organization
Death

An event assoicated with the death of an individual who has been diagnosed with COVID19
Lab Order Request Location

What is the location of the organization responsible for conducting the examination of the individual’s sample?
Admission Location

What is the location of the organization where the patient is currently being admitted to for treatment?
Lab Results

The result of the lab test which determines whether the patient is infected with COVID19 or not
Diagnosis

Confirmation of the diagnosis
Organization

Administering Organization
Immunization Tracking

Administer a suitable vaccine based on the most recent lab result

Structures: Extension Definitions

These define constraints on FHIR data types for systems conforming to this implementation guide.

Date Last Hospitalised

When last was the individual admitted to hospital?
Conditions or Comorbidities Present

Does the individual have any known pre-existing conditions and/or comorbidities?
Ever Hospitalized

Has the individual been admitted to hospital at any point in time in the past?
Date of next vaccination

When must the individual return to the organization for the next scheduled vaccination?

Terminology: Value Sets

These define sets of codes used by systems conforming to this implementation guide.

Test Result

COVID19 diagnosis
Lab Result Task Output

Custom code system to support the need to indicate that a COVID19 result was documented by the lab
Test Types

A list of COVID19 test types
Conditions / Comorbidity

List of conditions and comorbidities
Treatment Dispensed or Prescribed

A list of all possible drugs
Yes No Unknown

A list of options used to provide an answer
Cancellation Reasons

A list of reasons as a result of the lab task being cancelled
Vaccine Types

A list of COVID19 manufacturers
Reason for Assessment or Test Not Performed

A list of reasons associated with the service request for the lab order
Specimen Type

The type of specimen sample that was collected by the lab
Presentation

Determine if the COVID19 positive individual has any noticeable symptoms
Admission Types

A list of admission types
Patient Outcome

A list of possible outcomes.
Presenting symptoms

A list of symptoms
Source of Information

Custom code system to support the need to report on the origin of the individual’s COVID19 diagnosis
