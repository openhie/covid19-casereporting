Use Case Context

This use case supports integration between the EMR and lab systems in order to effectively process lab orders and have the results returned to the EMR for diagnosis.
In addition, this use case illustrates the need for a centralised data repository (DR) to enable regional and/or national COVID19 case reporting.

  • This use case diagram is limited to the specification associated with the design of this implementation guide.
  • For more information regarding the DR, please see the specification for the HAPI FHIR Datastore.
  • For more information regarding the implementation of the DR, please see the technical specificaton documentation.
